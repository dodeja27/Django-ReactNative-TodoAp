# Django-ReactNative-TodoAp
You must have virtual environment setup for Python to run the project
	
	$ pip install pipenv
	$ pipenv shell


To Run Django Server
	
	$ cd todoproject      	  	
	$ python manage.py runserver 0.0.0.0:8000
    
    
To Run Frontend

	$ cd frontend/TodoFrontend
	$ yarn install
	$ yarn start
